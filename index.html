ICCB Project Site

장내 마이크로바이옴 기반
대장암 예측 연구

공개 메타게놈 코호트를 이용하여 colorectal cancer prediction 모델을 비교하고, development cohort와 external validation cohort에서 일반화 성능을 평가하는 연구 프로젝트입니다.

다운로드 안내 이 공개 홈페이지에서는 데이터 파일을 직접 다운로드할 수 없습니다. 데이터 및 다운로드 가능한 파일은 Private GitHub Repository (ICCB-data) 에서만 관리합니다.
Public GitHub Repository Private Data Repository Project Homepage

Project Scope

공개 장내 메타게놈 데이터셋을 기반으로 CRC 예측 모델을 구축하고, Logistic Regression, Random Forest, XGBoost를 비교합니다.

Datasets

  • Development cohort: ZellerG_2014
  • External validation cohort: YachidaS_2019
