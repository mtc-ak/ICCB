프로젝트 개요

공개 가능한 코드, 문서, figure, 요약 결과를 한 곳에서 보여주는 소개용 사이트입니다. 데이터 배포보다는 연구 구조와 결과 해석을 빠르게 이해할 수 있도록 구성했습니다.

Datasets

  • Development cohort: ZellerG_2014
  • External validation cohort: YachidaS_2019

Models

  • Logistic Regression
  • Random Forest
  • XGBoost

핵심 결과

개발 코호트에서는 Random Forest가 가장 좋은 분류 성능을 보였고, 외부 검증에서는 XGBoost가 가장 높은 AUC를 보였습니다. 다만 외부 코호트에서 전체 성능은 감소했습니다.

Development Cohort 개발 코호트 테스트셋에서 Random Forest는 AUC 0.8487, F1-score 0.7500으로 가장 우수했습니다.
External Validation 외부 검증에서는 XGBoost가 AUC 0.6228로 가장 높았지만, 전체적으로 개발 코호트보다 성능이 하락했습니다.
Calibration 외부 코호트에서 ECE는 Random Forest가 0.0705로 가장 낮아 확률 보정 성능이 가장 좋았습니다.

해석 포인트

이 결과는 내부 테스트셋 성능만으로 모델의 실사용 가능성을 판단하기 어렵다는 점을 보여줍니다. 외부 검증에서 성능 저하가 나타났기 때문에, 마이크로바이옴 기반 CRC 예측모델의 일반화에는 추가 검증이 필요합니다.
