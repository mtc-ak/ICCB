프로젝트 개요

이 홈페이지는 데이터 다운로드용 포털이 아니라, 연구 구조와 분석 결과를 빠르게 이해할 수 있도록 구성한 공개 소개 페이지입니다. 공개 가능한 자료와 비공개 자료의 역할을 명확히 구분해 두었습니다.

사용 데이터셋

  • 개발 코호트: ZellerG_2014
  • 외부 검증 코호트: YachidaS_2019

비교 모델

  • Logistic Regression
  • Random Forest
  • XGBoost

공개 홈페이지 역할

코드, 논문 초안, 공개 figure, 요약 결과를 정리하여 연구 내용을 소개합니다.

비공개 저장소 역할

원본 데이터, 다운로드 파일, 대용량 산출물, 비공개 자료는 private 저장소에서만 관리합니다.

핵심 결과

개발 코호트에서는 Random Forest가 가장 높은 성능을 보였고, 외부 검증에서는 XGBoost가 가장 높은 AUC를 보였습니다. 다만 외부 검증에서 전체 성능은 감소해 일반화 한계를 함께 보여주었습니다.

개발 코호트 결과

개발 코호트 테스트셋에서 Random Forest는 AUC 0.8487, F1-score 0.7500으로 가장 우수했습니다.

외부 검증 결과

외부 검증에서는 XGBoost가 AUC 0.6228로 가장 높았지만, 전체적으로 개발 코호트보다 성능이 낮아졌습니다.

보정 성능

외부 코호트에서 ECE는 Random Forest가 0.0705로 가장 낮아 확률 예측의 안정성이 가장 좋았습니다.

결과 해석

내부 테스트셋에서 높은 성능이 나오더라도 독립적인 외부 코호트에서 같은 수준의 결과가 유지되지는 않았습니다. 따라서 마이크로바이옴 기반 대장암 예측모델은 유망하지만, 실제 일반화 가능성을 높이기 위한 추가 검증이 필요합니다.
