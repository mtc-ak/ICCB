ICCB Project

Microbiome-Based CRC Prediction

A public research project comparing machine learning models for colorectal cancer prediction using gut microbiome metagenomic cohorts and external validation.

Overview

This project evaluates microbiome-based prediction of colorectal cancer using public cohorts and compares Logistic Regression, Random Forest, and XGBoost.

Datasets

  • Development cohort: ZellerG_2014
  • External validation cohort: YachidaS_2019

Models

  • Logistic Regression
  • Random Forest
  • XGBoost
